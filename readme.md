# WordPress主题 商务风 简约风

### 主题介绍
wordpress主题这个东西网上可以说是遍地都是，但大多数都不是很满意；于是自己写一套吧。该主题主要特点是：商务风、简约、高效SEO
>SEO

该主题对SEO十分友好，针对文章等不同页面进行优化打造，更利于SEO爬虫获取

>代码高亮

支持绝大部分语言代码高亮

>友情链接

支持友情链接自定义

>文章浏览次数统计

加入文章浏览次数统计 无第三方插件

>文章点赞统计

加入文章点赞和踩的功能 无第三方插件 可在后台配置开关
